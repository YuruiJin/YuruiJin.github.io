这是一个使用 [tabs](https://github.com/Ovski4/jekyll-tabs) 实现标签页的文章示例。注意，标签页可用于多种目的，不仅限于代码展示。

## 第一个示例

要添加标签页，请使用以下语法：

{% raw %}

```liquid
{% tabs group-name %}

{% tab group-name tab-name-1 %}

Content 1

{% endtab %}

{% tab group-name tab-name-2 %}

Content 2

{% endtab %}

{% endtabs %}
```

{% endraw %}

使用此语法可以生成如下可视化效果：

{% tabs log %}

{% tab log php %}

```php
var_dump('hello');
```

{% endtab %}

{% tab log js %}

```javascript
console.log("hello");
```

{% endtab %}

{% tab log ruby %}

```javascript
pputs 'hello'
```

{% endtab %}

{% endtabs %}

## 另一个示例

{% tabs data-struct %}

{% tab data-struct yaml %}

```yaml
hello:
  - "whatsup"
  - "hi"
```

{% endtab %}

{% tab data-struct json %}

```json
{
  "hello": ["whatsup", "hi"]
}
```

{% endtab %}

{% endtabs %}

## 标签页的其他用途

{% tabs something-else %}

{% tab something-else text %}

普通文本内容

{% endtab %}

{% tab something-else quote %}

> 一段引用

{% endtab %}

{% tab something-else list %}

示例列表

- 定速自行车

{% endtab %}
